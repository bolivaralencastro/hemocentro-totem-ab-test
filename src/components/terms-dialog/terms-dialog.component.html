Termos de Uso

Este aplicativo é parte de um projeto pessoal e voluntário que eu, Bolívar Alencastro (Product Designer), estou desenvolvendo. Meu único objetivo é oferecer uma sugestão de melhoria de interface para o totem de autoatendimento do Hemocentro do Hospital das Clínicas de São Paulo.

Os dados que eu coleto são estritamente para minha análise de usabilidade e são compostos por:

  • Tempo de interação com cada uma das interfaces apresentadas.
  • Contagem de cliques nos links de hemocentros (sem identificação do usuário).

Sua privacidade é minha prioridade.

Fique tranquilo, nenhuma informação pessoal, como o CPF digitado, será armazenada ou enviada. Eu registro apenas o tempo de conclusão da tarefa, de forma totalmente anônima.

Sua participação neste teste é voluntária e anônima. Ao prosseguir, você concorda com a coleta não identificada dos dados de tempo de uso para os fins que descrevi acima.
